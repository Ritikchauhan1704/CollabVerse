# Tech stack
## Frontend

React 19

TailwindCSS

Shadcn

## Backend

Bun + Hono server

Neon pg

# Installation
git clone https://github.com/Ritikchauhan1704/CollabVerse.git

OR

gh repo clone Ritikchauhan1704/CollabVerse

Use [bun](https://bun.sh/) to install 
## Frontend

cd client

```bash
bun i
```
### Running
```bash
bun dev
```
## Backend

cd server

```bash
bun i
```

### Running
```bash
bun dev
```
